# elitebarber.io

Este proyecto consiste en el desarrollo de una landing page profesional para una barbería premium, diseñada con un enfoque moderno, elegante y centrado en la conversión de clientes.

La aplicación simula el sitio web real de una barbería en Buenos Aires, incluyendo presentación de servicios, equipo de trabajo, testimonios y un sistema de reservas, replicando la experiencia que tendría un usuario al buscar y agendar un turno online.

Actualmente, el formulario de contacto y reservas funciona a nivel frontend mediante JavaScript, junto con integración directa a WhatsApp, debido a las limitaciones de GitHub Pages, que no permite ejecutar lógica backend.

Sin embargo, la estructura del proyecto está pensada como si existiera un backend real, permitiendo escalar fácilmente a una implementación completa con base de datos, gestión de turnos y autenticación.

El objetivo principal del proyecto es demostrar habilidades en diseño UI, experiencia de usuario, desarrollo frontend y estructura escalable, siendo ideal para portafolios profesionales o proyectos freelance.

🧠 Lógica de negocio (concepto backend)

Aunque la implementación actual se realiza completamente en el frontend, la lógica simula un flujo real de negocio:

Captura de datos del cliente
Selección de servicio y profesional
Elección de fecha y horario
Validación de información
Simulación de envío de solicitud
Redirección opcional a WhatsApp

En un entorno real, esta lógica estaría conectada a un backend que permitiría:

Gestión de turnos en base de datos
Disponibilidad en tiempo real
Asignación automática de barberos
Notificaciones al cliente
Panel de administración
Historial de reservas

La arquitectura actual permite migrar fácilmente hacia tecnologías como Node.js, Firebase o Flask sin necesidad de rehacer el frontend.
